# Prosta gra łotropodbna

Jest to mój projekt na zaliczenie przedmiotu Programowanie gier łotropodobnych na UMCS Lublin.
Gra jest wyświetlana w ASCII oraz wykorzystuje bibliotekę curses do efektownego manipulowania terminalem w celu wyświetlenia gry.
Wykorzystanie tej biblioteki sprawia, że gra w terminalu wygląda płynnie oraz ładnie dzięki użyciu odpowiednich funkcji.
Więcej o samej grze i jej działaniu w pliku readme.md w folderze GDD_and_analyse.
